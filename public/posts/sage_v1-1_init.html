SAGE v1.1 — The Notebooks & Styles Update

The next step of Free Open Source AI Creation Engine.

Hello everyone!

It has been an intense journey since the first stable release of SAGE v1.0. I have been heads-down working on expanding what SAGE can do, specifically focusing on integrating new external notebooks to bring a massive array of new visual capabilities to your local environment.

For the SAGE v1.1 release, I want to take a step back from the technical details to focus on the community that makes this possible. While SAGE provides the orchestration and the engine, the fuel comes from the incredible open-source community.

🚀 Hybrid Power: External APIs & Generators

SAGE is not just about local generation. This release reinforces our hybrid architecture:

  • Pollinations.ai Integration: A true game-changer for SAGE users. We now fully support Pollinations for high-speed external image generation, allowing you to create without taxing your local hardware.
  • AI Text & Chat: SAGE now leverages external APIs for intelligent text operations, enabling AI-assisted prompt creation, brainstorming, and chat functionalities directly within the app.

What is SAGE?

SAGE (Storyboard Animation Generation Environment) is an open-source multimedia AI orchestration platform. It merges creative generation, development assistance, and intelligent orchestration under one cohesive system. Uniquely designed to run fully on Android via Termux, it also scales effortlessly to Linux and macOS environments.

SAGE transforms your device into a complete AI movie storyboard production studio. It orchestrates complex workflows via PHP and bash automation, manages projects through an intuitive web GUI, and now integrates new external notebooks for expanded stylistic capabilities. From local Stable Diffusion XL generation to high-speed external API calls via Pollinations, SAGE provides the tools to organize your visual narrative without technical barriers.
